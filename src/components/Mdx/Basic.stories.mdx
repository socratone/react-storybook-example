# Basic

`Badge` 컴포넌트 사용 방법

## Color

색을 변경할 수 있다.

<Canvas>
  <Story name="default color">
    <Badge>DEFAULT</Badge>
  </Story>
  <Story name="blue color">
    <Badge color="blue">BLUE</Badge>
  </Story>
  <Story name="red color">
    <Badge color="red">RED</Badge>
  </Story>
  <Story name="yellow color">
    <Badge color="yellow">YELLOW</Badge>
  </Story>
</Canvas>
